에브리타임 장터게시판에 원하는 물건이 올라올 때 바로 사지 못했을 때의 아쉬움이 있었다.

거기에 착안하여 프로젝트를 시작하게 되었다.

각 학교마다 url이 다르기 때문에 현재는 안동대학교 에브리타임만 작동한다.
